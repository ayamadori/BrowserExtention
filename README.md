BrowserExtention (now call URLTranslator)
=====================================

A helper for website for Nokia Asha Platform.   

Feature 
--------- 
* Translate website 
* Convert website to Mobile View
* Use from [Share] menu

Use of Nokia APIs and plug-ins
--------------------
* Share API
* Tantalum plug-in

Test
-------
Tested on Nokia Asha SDK 1.1 & 1.2(beta) and Asha 230.
